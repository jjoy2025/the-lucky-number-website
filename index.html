Game Rules

This game is based on Single Numbers and Multi Numbers.

  • Multi Number: Any three-digit number chosen from the provided list of candidates.

How results are calculated:
If the Single Number result is 3, then the Multi Number will be something like 580 or 346.

For example:
5 + 8 + 0 = 13 → Single digit = 3
3 + 4 + 6 = 13 → Last digit = 3

The sum of all digits is calculated, and if it matches the Single Number, that is the winning number. If a double-digit result appears, the last digit or last digit will be considered the winning number.
